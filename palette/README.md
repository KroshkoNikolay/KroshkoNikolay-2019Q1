# CodeJam-dom Palette

## Start project

1. Clone project ```git clone https://github.com/rolling-scopes-school/KroshkoNikolay-2019Q1.git```
2. Install dependencies ```npm i```
3. Open `index.html` file in browser

## Run eslint tests

```npm test```
